# Mio 澪
_✨ 基于 Nonebot2 的超可爱QBot ✨_
</div>

## 简介
澪(Mio) 基于Nonebot2 开发，用于群聊的机器人

## 功能简介
- 插件管理
- 插件帮助
- AI聊天（基于百度api）
- 扫雷
- 猜单词

## 鸣谢
